                How to play

                Add players (or join a room), then draw cards. On your turn: read the card aloud and either do it / answer honestly, or take the Drink / Pass option. Then it's the next player.

                The levels

                • L1 Icebreaker — light, funny warm-ups.
                • L2 Spicy — flirty and bold, still tame.
                • L3 Freaky — bolder dares for grown-ups.
                • L4 OMG — maximum chaos.
                • 🃏 Wildcards — swaps, group votes & chaos mechanics you run yourselves.

                Modes

                Solo / Pass the Phone — one device, pass it around the circle.

                Multiplayer — one person creates a room; everyone else joins with the code and every phone stays in sync.

                Make it yours

                Everything's on by default — switch off anything you'd rather skip in ⚙ Game settings. Write your own prompts and group them into categories under Custom Cards.

                One rule

                If you don't want to do a card, just take the Drink / Pass and move on — no big deal. Drink responsibly (works just as well with a soft drink).
